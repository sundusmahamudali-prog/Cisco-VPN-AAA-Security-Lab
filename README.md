# 🌐 VPN Implementation with AAA Authentication
This repository contains a comprehensive networking lab focused on the deployment of a **Virtual Private Network (VPN)** and **AAA (Authentication, Authorization, and Accounting)** services. 

Completed as part of my networks studies at **Rathmines College**.

## 🚀 Project Overview
The goal of this lab was to create a secure tunnel between remote clients and a central server, ensuring all traffic is encrypted and users are authenticated via a dedicated AAA Radius server.



## 🛠️ Technical Specifications
* **Protocols:** AAA (Radius), VPN Tunneling, TCP/IP, ICMP.
* **Security Features:** * **AAA Server:** Centralized user tracking and access control.
    * **Encryption:** WEP and VPN tunneling for data privacy.
    * **Layer 2 Security:** MAC Address filtering (Port Security).
* **Software:** Cisco Packet Tracer v6.2.

## 📝 Key Learning Outcomes
* **Secure Routing:** Configured static routes to ensure traffic is forced through the VPN gateway rather than the local switch.
* **Encapsulation:** Observed how data is wrapped in a secure "tunnel" during simulation mode.
* **Troubleshooting:** Identified and resolved initial packet loss caused by network convergence and ARP.

## 📂 Files in this Repository
* **Assignment Report (PDF):** A 17-page detailed walkthrough with screenshots of every configuration step and simulation result.
* **Packet Tracer (.pkt):** The functional simulation file for live testing.
